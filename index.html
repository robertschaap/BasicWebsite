About Fonts Tested

This is a simple site about fonts. There are a few CSS properties that have a big impact on how your site is presented and behaves. Two of these are the font-size and font-weight properties.

Font Sizes

There are three main ways to size things in CSS; using pixels, EM units and REM units. Depending on which you choose to use for your site, you will need to be aware of a few caveats. The font sizes page dives into these and shows a few examples.

Font Weights

No doubt you're aware of the bold, italic and underline styles that are found in text editors, e-mail programmes and basically any kind of program that allows you to format text. The bold property is an example of font weight in CSS, it allows you to style how "heavy" a font looks. The font weight page shows you how to use CSS' most common properties.
